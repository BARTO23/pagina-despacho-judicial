# The Sovereign Ledger - Landing Page para Abogado Especialista ⚖️

![Vite](https://img.shields.io/badge/vite-%23646CFF.svg?style=for-the-badge&logo=vite&logoColor=white)
![React](https://img.shields.io/badge/react-%2320232a.svg?style=for-the-badge&logo=react&logoColor=%2361DAFB)
![TailwindCSS](https://img.shields.io/badge/tailwindcss-%2338B2AC.svg?style=for-the-badge&logo=tailwind-css&logoColor=white)

Una landing page moderna, minimalista y de **nivel editorial** diseñada específicamente para **Juan Felipe Restrepo Sánchez**, Abogado Especialista en Derecho Penal y Procesal Penal.

Este proyecto fue desarrollado bajo una visión de diseño ("The Sovereign Ledger") que busca evocar autoridad absoluta, precisión y sofisticación de alta gama, rompiendo con el clásico tema recargado de los directorios legales utilizando transiciones suaves, tipografía de peso y un sistema de color "Deep Navy" y "Cerulean Blue".

## 🚀 Características Principales

- **Diseño Editorial & Asimétrico:** Maquetación limpia, con vasto espacio en blanco como estructura.
- **Glassmorphism & Tonal Layering:** Barras de navegación flotantes difuminadas y capas tonales que añaden profundidad sin usar bordes rígidos.
- **Micro-interacciones Fluidas:** Equipado con `react-magic-motion` para animaciones y expansiones de tarjetas completamente orgánicas y libres de saltos bruscos.
- **Optimización de Rendimiento:** Construido sobre **Vite** y **React 18** para tiempos de carga y respuesta de interacción casi instantáneos.
- **Sistema de Tokens CSS Personalizado:** Integrado profundamente con un motor iterativo de Tailwind CSS para garantizar la paleta de colores corporativa (Navy/Cerulean).

## 🛠 Entorno Moderno de Desarrollo

- **Frontend Core:** [React 18](https://react.dev/) + [Vite](https://vitejs.dev/)
- **Estilos:** [Tailwind CSS v3](https://tailwindcss.com/)
- **Animaciones Automáticas:** [react-magic-motion](https://react-magic-motion.com/)
- **Íconos:** Material Symbols Outlined

## 🏗 Cómo ejecutarlo de forma local

Si deseas clonar o correr el repositorio en tu propia máquina, sigue estos pasos:

1. **Asegúrate de tener Node.js instalado.**
2. **Instala las dependencias del proyecto:**
   ```bash
   npm install
   ```
3. **Inicia el servidor de desarrollo:**
   ```bash
   npm run dev
   ```
4. Abre [http://localhost:5173](http://localhost:5173) para verlo en tu navegador local.

## 🎨 Paleta de Colores

Gobernado bajo una paleta estricta basada en su tarjeta de presentación personal:

- **Deep Navy Blue:** `#000133` (Autoridad, Hero background)
- **Cerulean Blue:** `#00658b` (Claridad, Acentos y botones fantasma)
- **Superficies Claras:** Variantes desde `#fcf8fd` hasta `#ffffff` (Bases de lectura)
- **Metálicos:** Detalles en `#777680` y escalas de gris.

## 📱 Responsividad Garantizada
La arquitectura se estructuró contemplando dispositivos de gran resolución (modo _Desktop_ prioritario mediante diseño horizontal/side-by-side de Stitch), mientras Tailwind se encarga de acoplarlo limpiamente para pantallas pequeñas mediante el sistema Flexbox y Grid.
